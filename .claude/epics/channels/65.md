# Task: TwilioSmsAdapter + signed webhook + 503 fallback

## Description

Second concrete adapter plus the `POST /api/webhooks/twilio/sms/` endpoint. Verifies Twilio signature using `TWILIO_AUTH_TOKEN`. When the token is unset, returns `503 not_configured` cleanly.

## Acceptance Criteria

- [ ] Valid signed payload → 200 with a TwiML reply containing the assistant's response.
- [ ] Tampered signature → 403.
- [ ] `TWILIO_AUTH_TOKEN` unset → 503 `{ "error": "not_configured" }`, no test failure, no traceback.
- [ ] Conversation row stored with `channel='sms'` and the inbound `From` phone number as `customer_identifier`.

## Technical Details

- `backend/agent/channels/twilio_sms.py`.
- `backend/api/webhooks/twilio_sms.py` for the DRF view.
- Signature verification via `twilio.request_validator.RequestValidator`.

- 64 — base adapter must exist.
