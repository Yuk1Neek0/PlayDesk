# Task: Public landing page /qr/[slug] (SSR + bilingual + branded)

## Description

Server-rendered page at `/qr/[slug]` that shows the store's brand mark + accent, the configured `QRAction` chips in `position` order, and fires a `scan` event before rendering. Bilingual via `?lang=zh` or `Accept-Language`.

## Acceptance Criteria

- [ ] Page renders in <1.5s under DevTools Fast 3G against seed data.
- [ ] Disabled actions are not displayed.
- [ ] Chip click fires `navigator.sendBeacon('/api/qr/event/', {...})` and immediately navigates to `target_url`.
- [ ] Chinese visitors (Accept-Language `zh*`) see Chinese labels for built-in `kind`s; custom labels are rendered as-typed.
- [ ] Lighthouse Performance ≥90 on the public page.

## Technical Details

- `frontend/src/app/qr/[slug]/page.tsx` — server component, fetches actions + brand via a backend helper endpoint (`GET /api/qr/{slug}/`).
- Reuses `pd-chip` and `pd-brand-mark` styles.
- Falls back to `pd-page-head` placeholder when the store has no `brand.logo_url`.
