# Task: Chat page

## Description

Turn the `/chat` shell into a streaming AI front-desk chat: message composer +
transcript, consuming the SSE stream via the hook from task 001. Renders
streaming assistant tokens and in-flight tool-call hints ("checking
availability…", "looking up policy…").

## Acceptance Criteria
- [ ] Message composer sends a user message and starts an SSE stream.
- [ ] Assistant tokens render incrementally as they arrive.
- [ ] In-flight tool-call hints display while tools run and clear when done.
- [ ] The UI never freezes during long tool-call sequences.
- [ ] Component tests pass against a mocked SSE stream.
- [ ] CI green: lint, typecheck, vitest, build.

## Technical Details

- Use the SSE hook from task 001; non-blocking token rendering.
- Tailwind defaults only.
- Touches only the `/chat` route files — conflict-free with tasks 002 and 004.

## Dependencies

- Task 001 (SSE hook).
