# Task: Manual booking page

## Description

Wire the `/` page shell into a working manual booking flow: resource picker →
date → live availability slots → confirm. Calls the real availability and
booking endpoints through the generated client from task 001.

- [ ] Component tests pass against mocked API responses.
- [ ] CI green: lint, typecheck, vitest, build.
