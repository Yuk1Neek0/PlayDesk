# Task: Admin config page /admin/qr — drag-and-drop + analytics cards

## Description

Staff-gated page at `/admin/qr` that lists configured actions, supports inline edit, drag-and-drop reorder (keyboard-accessible), and shows a top-of-page card row of analytics (scans, clicks, engagement rate, per-action breakdown).

## Acceptance Criteria

- [ ] Page lists actions in `position` order with inline-edit on label / URL / points / enabled.
- [ ] Drag-and-drop reorder calls PATCH with the new `position`; row reflects the new order on success, reverts on error.
- [ ] Keyboard: Tab focuses a row, Arrow Up / Arrow Down reorders it (calls the same PATCH).
- [ ] Top-of-page card row uses `<MetricCard>` and shows last 7-day analytics by default, with `7 | 30 | 90` filter chips.
- [ ] "Preview" button opens `/qr/[slug]` in a new tab.
- [ ] Auth gate redirects non-staff to `/login`.

## Technical Details

- `frontend/src/app/admin/qr/page.tsx`.
- `frontend/src/components/qr-action-row.tsx` for one editable row.
- `frontend/src/components/metric-card.tsx` (shared with future analytics work).
- Drag implementation: HTML5 native drag-and-drop with a small custom hook — avoid pulling in a 3rd-party DnD library.
