# Task: Bilingual retrieval

## Description

Make the assistant bilingual (EN / 中): detect the user message's language,
filter RAG retrieval to knowledge-base chunks of that language, and instruct
the LLM to answer in the user's language.

## Acceptance Criteria
- [ ] User-message language is detected (EN vs ZH).
- [ ] RAG retrieval filters `KnowledgeChunk` by the detected `lang`.
- [ ] The system prompt instructs the LLM to reply in the user's language.
- [ ] A Chinese query is answered in Chinese from `lang: zh` chunks.
- [ ] Tests cover both EN and ZH paths.
- [ ] CI green: `ruff check`, `ruff format`, `pytest`.
