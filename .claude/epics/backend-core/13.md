# Task: Hand-rolled agent loop

## Description

Implement the hand-rolled agent loop in a new `backend/agent/` app, dispatching tools via the frozen registry.

## Acceptance Criteria
- [ ] Context assembly: system prompt + top-k RAG chunks + last N `Message` rows.
- [ ] LLM call behind an injectable Claude client interface (mockable in tests).
- [ ] If the response has tool calls: execute via the registry (parallel where safe), append structured results, iterate.
- [ ] If plain text: finalize.
- [ ] Hard cap at 6 iterations with a graceful human-handoff fallback.
- [ ] Every turn (user / assistant / tool call / tool result) persisted as a `Message` row; tool payloads as JSONB.
- [ ] LLM failure → exponential backoff, max 3 retries; tool failure → structured error back to the LLM, never to the user.
- [ ] pytest drives the loop with a mocked LLM, asserting tool dispatch, iteration cap, and persistence.

## Technical Details
- All code under `backend/agent/`; do NOT edit `config/urls.py`/`config/settings.py` — report what is needed.
- Build against the frozen tool registry interface — does not require task 003's real bodies to be complete.

## Dependencies
Frozen tool registry + schemas (foundation).
